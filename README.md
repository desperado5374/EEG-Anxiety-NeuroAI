# EEG-Anxiety-NeuroAI

NeuroAI-based EEG signal analysis pipeline for anxiety prediction using machine learning and deep learning approaches.

## Project Goals

- EEG preprocessing
- Artifact removal
- Feature extraction
- Deep learning for anxiety prediction
- Multimodal NeuroAI modeling

## Current Progress

### Completed
- EEG data loading using MNE
- Bandpass filtering
- Notch filtering
- EEG visualization
- Window segmentation
- Statistical feature extraction

## Tools & Libraries

- Python
- MNE
- NumPy
- Scikit-learn
- TensorFlow / PyTorch

## Future Work

- CNN/LSTM models
- Attention-based EEG analysis
- Explainable AI (SHAP / Grad-CAM)
- Multimodal fusion with physiological data
